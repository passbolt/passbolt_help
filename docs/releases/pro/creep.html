    v4.0.2 ~ Creep (pro)

    Release date: June 1st, 2023.

    Presenting the latest update of passbolt Pro, version 4.0.2. This release revises a previous fix for a community-reported bug which impacts 32-bit operating systems. It also includes a fix for a bug where the triage page wasn’t displayed if SSO was misconfigured.

    Find out more about the reported issue on the community forum.

    API

    Fixed

    • PB-24644 As an admin I should be able to run migrations on a 32 bit environment
    • PB-24657 As a user I should see the triage page even when SSO is misconfigured
